## 3. 주요 기능 요약

### 3.1 정보탐색 모듈

**구현 파일**: `frontend/src/pages/Explore.tsx`

**주요 기능**
- AI 기반 질문답변 (OpenAI GPT-4o-mini)
- 뉴스 요약 (네이버 뉴스 API 통합)
- 키워드 자동 추출
- 점자 출력 (하드웨어 연동)
- 복습 목록 자동 저장

**API 엔드포인트**
- `GET /api/chat/explore/?q={query}` - 정보탐색 통합
- `POST /api/chat/ask/` - 질문답변
- `POST /api/chat/detail/` - 자세한 설명

### 3.2 점자학습 모듈

**구현 파일**
- `frontend/src/pages/LearnIndex.tsx` - 학습 메뉴
- `frontend/src/pages/LearnStep.tsx` - 학습 단계
- `frontend/src/pages/Quiz.tsx` - 퀴즈
- `frontend/src/pages/Review.tsx` - 복습

**주요 기능**
- 자모/단어/문장 단계별 학습
- 점자 패턴 시각화
- 음성 입력 퀴즈
- SRS 기반 복습 시스템
- 자유 변환 모드

**API 엔드포인트**
- `GET /api/learn/chars/` - 자모 학습 데이터
- `GET /api/learn/words/` - 단어 학습 데이터
- `GET /api/learn/sentences/` - 문장 학습 데이터
- `GET /api/learning/list/` - 복습 항목 목록
- `POST /api/learning/save/` - 복습 항목 저장
- `POST /api/learning/grade/{item_id}/` - 복습 결과 제출

### 3.3 음성 제어 시스템

**구현 파일**
- `frontend/src/components/input/GlobalVoiceRecognition.tsx`
- `frontend/src/hooks/useSTT.ts`
- `frontend/src/hooks/useTTS.ts`
- `frontend/src/lib/voice/CommandRouter.ts`
- `frontend/src/lib/voice/MisrecognitionMap.ts`

**주요 기능**
- Confidence 기반 필터링
- 오인식 패턴 보정
- 컨텍스트 기반 우선순위
- 사용자 피드백 수집

**지원 명령어**
- 네비게이션: "홈", "뒤로", "학습", "탐색", "복습", "자유변환"
- 제어: "다음", "이전", "반복", "정지", "시작"
- 학습 메뉴: "자모", "단어", "문장"

### 3.4 하드웨어 연동

**구현 파일**
- `frontend/src/hooks/useBrailleSerial.ts` - Web Serial API
- `frontend/src/hooks/useBrailleBLE.ts` - Web Bluetooth API
- `arduino/braille_3cell/braille_3cell.ino` - Arduino 펌웨어
- `raspberrypi/ble_server.py` - BLE 서버

**주요 기능**
- Web Serial API 직접 연결 (권장)
- Web Bluetooth API (Raspberry Pi 브릿지)
- 3셀 점자 디스플레이 제어
- 점자 패턴 실시간 출력

## 8. 개발 이슈 및 해결

### 8.1 주요 이슈

#### 이슈 1: 음성 인식 오인식 문제

**문제**
- "자모"가 "자무"로 인식
- "디귿"이 "디긋"으로 인식
- 학습 메뉴에서 명령어 인식률 낮음

**해결**
- 오인식 패턴 사전 구축 (`MisrecognitionMap.ts`)
- Confidence 기반 필터링
- 컨텍스트 기반 우선순위 적용

**결과**
- 음성 인식 정확도 90% 이상 달성

---

#### 이슈 2: 학습 메뉴에서 "학습" 명령 처리

**문제**
- 학습 메뉴에서 "학습" 명령 시 자모 학습으로 직접 이동
- 사용자가 메뉴 선택을 원함

**해결**
- `LearnIndex.tsx`에서 `learn` 핸들러 제거
- `Home.tsx`에서만 "학습" 명령 처리
- 학습 메뉴에서는 하위 메뉴만 선택 가능

**결과**
- 직관적인 네비게이션 구현

---

#### 이슈 3: 음성 인식 속도 문제

**문제**
- 학습 메뉴에서 음성 인식이 느림
- Alternatives 처리로 인한 지연

**해결**
- 기본 텍스트 우선 처리
- Alternatives는 confidence 높은 순으로 처리
- 조기 종료 로직 적용

**결과**
- 음성 인식 응답 속도 < 1초 달성

---

#### 이슈 4: 한글 점자 변환

**문제**
- UTF-8 인코딩 처리
- 초성/중성/종성 분해
- 점자 패턴 매핑

**해결**
- `backend/utils/encode_hangul.py` 구현
- `backend/data/ko_braille.json` 매핑 데이터 구축
- 3셀 점자 표시 로직 구현

**결과**
- 한글 점자 변환 100% 정확도 달성

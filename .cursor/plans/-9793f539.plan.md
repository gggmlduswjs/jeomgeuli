# 하드웨어 연동 구현 계획

## 목표

HARDWARE_SPEC.md에 명시된 스펙에 따라 점글이 하드웨어 시스템을 완전히 연동합니다.

## 구현 범위

### 1. 프론트엔드 수정

- `useBrailleBLE.ts`: 점글이 하드웨어 설정 추가
- `Explore.tsx`: 점글이 하드웨어 사용하도록 수정

### 2. Raspberry Pi BLE 서버 생성

- `raspberrypi/` 폴더 생성
- `raspberrypi/ble_server.py`: BLE → Serial Bridge 서버 구현
- `raspberrypi/README.md`: 설치 및 실행 가이드

### 3. Arduino 펌웨어 생성

- `arduino/` 폴더 생성
- `arduino/braille_firmware.ino`: 점자 모듈 제어 펌웨어
- `arduino/README.md`: 업로드 및 사용 가이드

### 4. 문서 업데이트

- `README.md`: 하드웨어 연동 섹션 추가

1. `raspberrypi/` 폴더 생성

- Serial 통신 설정 (115200 baud)
- BLE → Serial Bridge 로직

3. `raspberrypi/README.md` 생성

- 설치 방법 (bluezero, pyserial)

- 문제 해결 가이드

- 점자 패턴 변환 로직 (기본 구조)

## 기본 사용법

### 1. 음성 인식 시작하기
- **마이크 버튼 클릭**: 화면의 마이크 아이콘 버튼을 클릭합니다
- **음성 명령**: "음성 시작" 또는 "음성 입력" (일부 페이지에서 지원)
- **자동 시작**: Home 페이지 진입 시 자동으로 음성 인식이 활성화됩니다

### 2. 음성 명령 말하기
- 마이크가 활성화되면 명령어를 명확하게 말합니다
- 브라우저가 마이크 권한을 요청하면 "허용"을 선택합니다
- 조용한 환경에서 사용하면 인식률이 높아집니다

### 3. 음성 인식 중지하기
- **"멈춰"**, **"정지"**, **"스탑"**, **"일시정지"**, **"중지"** 명령으로 중지
- 마이크 버튼을 다시 클릭하여 중지
- ESC 키 누르기 (일부 페이지에서 지원)

## 페이지별 음성 명령어

### 🏠 홈 화면 (Home)

#### 네비게이션
- **"홈"**, **"메인"**, **"처음으로"**, **"홈으로"** → 홈 화면 유지
- **"뒤로"**, **"이전"**, **"뒤로가기"** → 이전 페이지로 이동

#### 페이지 이동
- **"학습"**, **"학습하기"**, **"공부"**, **"점자 학습"** → 점자 학습 모드로 이동
- **"정보탐색"**, **"탐색"**, **"검색"**, **"정보"** → 정보 탐색 모드로 이동
- **"복습"**, **"리뷰"**, **"다시 보기"** → 복습 모드로 이동
- **"자유변환"**, **"변환"**, **"점자 변환"** → 자유 변환 모드로 이동
- **"퀴즈"**, **"문제"**, **"테스트"**, **"시험"** → 퀴즈 모드로 이동

#### 기타
- **"도움말"**, **"도움"**, **"헬프"**, **"사용법"**, **"명령어"** → 사용 가능한 명령어 안내

---

### 🔍 정보 탐색 (Explore)

#### 기본 제어
- **"멈춰"**, **"정지"**, **"스탑"** → 음성/점자 출력 중지
- **"홈"**, **"메인"** → 홈으로 이동
- **"뒤로"**, **"이전"** → 이전 페이지로 이동

#### 점자 제어
- **"점자 출력 켜"**, **"점자 켜"**, **"점자 시작"**, **"점자 활성화"** → 점자 출력 활성화
- **"점자 출력 꺼"**, **"점자 꺼"**, **"점자 중지"**, **"점자 비활성화"** → 점자 출력 비활성화
- **"점자 연결"**, **"점자 디스플레이 연결"**, **"블루투스 연결"** → 점자 디스플레이 연결
- **"점자 해제"**, **"점자 디스플레이 해제"**, **"블루투스 해제"** → 점자 디스플레이 연결 해제

#### 재생 제어
- **"다음"**, **"넘겨"**, **"다음으로"**, **"계속"**, **"진행"** → 다음 점자 셀로 이동
- **"반복"**, **"다시 말해"**, **"다시 읽어"**, **"재생"** → 현재 점자 반복 재생
- **"시작"**, **"시작해"**, **"재개"**, **"계속해"** → 점자 재생 시작

#### 정보 탐색
- **"뉴스"**, **"뉴스 보기"**, **"오늘 뉴스"** → 오늘 뉴스 검색
- **"날씨"**, **"날씨 보기"**, **"오늘 날씨"** → 오늘 날씨 검색
- **"자세히"**, **"더 알려줘"**, **"자세하게"**, **"상세히"** → 마지막 키워드에 대한 상세 정보 요청

#### 텍스트 입력 및 전송
1. 마이크 버튼 클릭하여 음성 입력 시작
2. 질문을 말합니다 (예: "오늘 날씨 알려줘", "비트코인에 대해 설명해줘")
3. **"전송"**, **"제출"**, **"확인"**, **"입력"** 명령으로 자동 전송
   - 또는 질문 끝에 "전송"을 붙여 말하기 (예: "오늘 날씨 알려줘 전송")

#### 입력 제어
- **"지워"**, **"삭제"**, **"초기화"**, **"다시 입력"** → 입력창 초기화

### 📚 퀴즈 (Quiz)

#### 기본 제어
- **"홈"** → 홈으로 이동
- **"뒤로"** → 학습 모드로 이동
- **"멈춰"**, **"정지"** → STT 중지

#### 문제 풀이
1. **음성으로 정답 말하기**:
   - 마이크 버튼 클릭
   - 정답을 말합니다 (예: "기역", "가나다")
   - 자동으로 제출되거나 **"전송"**, **"제출"** 명령으로 제출

2. **음성 명령으로 제출**:
   - 정답을 입력한 후 **"전송"**, **"제출"**, **"확인"**, **"입력"** 명령

#### 입력 제어
- **"지워"**, **"삭제"**, **"초기화"**, **"다시 입력"** → 입력 초기화
- **"다음"** → 정답 제출 후 다음 문제로 이동
- **"반복"** → 문제 다시 읽기

## 음성 입력 방법

### 방법 1: 마이크 버튼 사용
1. 화면의 마이크 아이콘 버튼 클릭
2. 명령어 또는 텍스트 말하기
3. 자동 인식 또는 "전송" 명령으로 실행

### 방법 2: 음성 명령 포함
- 질문 끝에 명령어를 붙여 말하기
  - 예: "오늘 날씨 알려줘 전송"
  - 예: "비트코인 설명해줘 제출"

### 방법 3: 연속 명령
- Home 페이지에서는 연속으로 여러 명령을 말할 수 있습니다
- 예: "정보탐색" → "뉴스" → "자세히"

## 자주 묻는 질문

### Q1: 마이크 권한이 거부되었어요
**A**: 브라우저 설정에서 마이크 권한을 허용해야 합니다.
- Chrome: 주소창 왼쪽의 자물쇠 아이콘 클릭 → 사이트 설정 → 마이크 → 허용
- Edge: 설정 → 사이트 권한 → 마이크 → 허용

### Q2: 음성이 인식되지 않아요
**A**: 
- 조용한 환경에서 사용하세요
- 마이크가 제대로 연결되어 있는지 확인하세요
- 브라우저가 마이크를 사용 중인지 확인하세요 (다른 탭에서 사용 중일 수 있음)
- "음성 인식 오류" 메시지가 나오면 페이지를 새로고침하세요

### Q3: 음성 명령이 작동하지 않아요
**A**:
- 명령어를 정확하게 말했는지 확인하세요
- Home 페이지에서는 연속 음성 인식이 활성화되어 있어야 합니다
- 다른 페이지에서는 마이크 버튼을 클릭한 후 명령어를 말해야 합니다

### Q4: 점자 디스플레이 연결이 안 돼요
**A**:
- Bluetooth가 켜져 있는지 확인하세요
- 점자 디스플레이가 켜져 있고 페어링 모드인지 확인하세요
- 브라우저가 Web Bluetooth API를 지원하는지 확인하세요 (Chrome, Edge 권장)
- HTTPS 환경에서만 작동합니다 (localhost는 예외)

### Q5: 음성 입력 후 자동으로 전송되지 않아요
**A**:
- "전송", "제출", "확인", "입력" 명령을 명시적으로 말하세요
- 또는 Enter 키를 누르거나 전송 버튼을 클릭하세요

---

## 💡 사용 팁

1. **명확하게 말하기**: 조용한 환경에서 명확하게 발음하면 인식률이 높아집니다
2. **명령어 기억하기**: 자주 사용하는 명령어를 외워두면 편리합니다
3. **음성 안내 활용**: "도움말" 명령으로 사용 가능한 명령어를 들을 수 있습니다
4. **단계별 사용**: 복잡한 작업은 단계별로 나누어 명령하세요
5. **재시도**: 인식이 안 되면 다시 말해보세요
